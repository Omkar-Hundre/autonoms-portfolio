# Autonoms

A premium, modern portfolio website for Autonoms, a specialized agency building real production systems.

## What we do
- **Web Applications**: Real systems — with logins, dashboards, real-time data, and actual users.
- **Mobile Apps**: Android/iOS apps with push notifications, offline mode, and hardware integrations.
- **AI & Automation**: Scraping, workflow bots, LLM pipelines, and computer vision.
- **Desktop Software**: Offline-first applications built for industrial and enterprise environments.
- **CRM & Operations**: Quotation systems, project trackers, and lead pipelines.
- **Design & Branding**: Logos, brand identity, content, and video editing.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Vanilla CSS & TailwindCSS (Utility classes)
- **Deployment**: GitHub Pages
- **Animations**: Custom IntersectionObserver (`useReveal`)

## Local Development
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. The site will be running at `http://localhost:5173`
